# giv_tcp

GivEnergy solar inverter monitoring via
[GivTCP](https://github.com/GivEnergy/givtcp).

Reads data from a GivEnergy inverter at a fixed LAN IP and publishes
metrics to MQTT (consumed by Home Assistant).

> **Note:** This directory currently contains no job file — the job may be
> defined elsewhere or not yet migrated. Check git history or the running
> cluster for the current state.
